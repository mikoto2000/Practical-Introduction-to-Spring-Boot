# Spring Boot 実践入門

## 概要

日本において、 Java のフレームワークと言えば Spring Boot がほぼデファクトスタンダードとなっており、業務で触れずに済むことはほとんどありません。
普段あまり意識せずに使っている機能について、なぜそう書くのか／どこまで気にすべきかをコード例で整理します。
明日からの実装やレビューが少し楽になることを目指します。
