# このワークショップについて

## やること

JSON を返却するミニマムな API を例に、プロジェクト構成・作り方について説明します。

- プロジェクトの作り方(Spring Initializr)
- シンプルな API の実装

    - bean について
    - Controller -> Service -> Util と呼び出す API 構成
- application.yaml の機能いろいろ
    - プロファイル
    - 環境変数読み込み
- jar デプロイ

## やらないこと
